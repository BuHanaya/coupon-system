# نظام إدارة الكوبونات 🎟️

موقع كامل لتوليد وإدارة والتحقق من كوبونات الشراء — يعمل على أي متصفح بدون سيرفر.

## المميزات
- **توليد كوبونات** بأرقام تسلسلية تلقائية
- **إدارة كاملة** مع بحث وفلترة
- **التحقق الفوري** من صلاحية الكوبون وتسجيل استخدامه
- **إحصائيات** شاملة
- **حفظ تلقائي** في المتصفح (localStorage)
- **تسجيل دخول** بكلمة مرور

---

## 🚀 كيف ترفعه على GitHub Pages

### الخطوة 1 — إنشاء Repository
1. افتح [github.com](https://github.com) وسجل دخول
2. اضغط **New repository** (الزر الأخضر)
3. اسم المشروع: `coupon-system` (أو أي اسم تريده)
4. اتركه **Public**
5. اضغط **Create repository**

### الخطوة 2 — رفع الملفات
1. في صفحة الـ repository، اضغط **uploading an existing file**
2. اسحب الملفات الثلاثة:
   - `index.html`
   - `style.css`
   - `app.js`
3. اضغط **Commit changes**

### الخطوة 3 — تفعيل GitHub Pages
1. اذهب لـ **Settings** في الـ repository
2. من القائمة الجانبية، اضغط **Pages**
3. تحت **Source**، اختر **Deploy from a branch**
4. اختر **main** branch و **/ (root)**
5. اضغط **Save**

### الخطوة 4 — الرابط
بعد دقيقتين، سيظهر رابط الموقع:
```
https://[اسم-حسابك].github.io/coupon-system/
```

## 🔑 تغيير كلمة المرور
افتح ملف `app.js` وغيّر السطر:
```javascript
const PASSWORD = 'admin123';
```
مثال: `const PASSWORD = 'mypassword2024';`

## ⚠️ ملاحظة مهمة
البيانات محفوظة في **localStorage** — تعني:
- ✅ تبقى حتى لو أغلقت المتصفح
- ✅ تشتغل بدون إنترنت بعد التحميل الأول
- ⚠️ البيانات مختلفة بين الأجهزة (كل جهاز له بياناته)
- ⚠️ لو مسحت cache المتصفح، تحذف البيانات

**لو تبي البيانات مشتركة بين أجهزة** → نضيف قاعدة بيانات سحابية لاحقًا.
